# 🍳 Cook_Book | Flutter Recipe Application

A feature-rich **Cookbook Application** built with Flutter. This app allows users to explore recipes, manage ingredients, and view cooking instructions with a modern and responsive UI.

---

## 🚀 Key Features

* **📖 Recipe Catalog**: Browse through a diverse collection of recipes.
* **📂 Categorized View**: Recipes organized by types for easy navigation.
* **🖼️ Rich Media Support**: High-quality images for each dish.
* **🛠️ Advanced Architecture**: Uses a clean folder structure for scalability.
* **🎨 Material 3 Design**: Beautifully crafted UI with smooth transitions.

---

## 📂 Project Architecture

This project follows a professional directory structure:

```text
lib/
├── models/         # Data blueprints for Recipes & Ingredients
├── screens/        # UI Screens (Home, Details, Categories)
├── services/       # API calls or Local Storage logic
├── utils/          # Constants, Themes, and Helper functions
├── widgets/        # Reusable UI components (Cards, Buttons)
└── main.dart       # App entry point
🛠️ Tech Stack
Frontend: Flutter & Dart

State Management: Provider / SetState (Base implementation)

Data Handling: JSON / Models Mapping

UI Icons: Cupertino & Material Icons
